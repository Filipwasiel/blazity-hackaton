# Project Vocabulary

Canonical product and codebase terms for AI agents. See [[product]] for context.

## Terms

| Term | Meaning | Avoid |
| --- | --- | --- |
| Idea | The single raw input — a sentence or short paragraph the user types in to expand. | prompt (reserve for the Claude prompt), topic |
| Format | One of the six platform-tailored outputs generated from an idea. | output type, variant |
| The six formats | Tweet, LinkedIn post, newsletter intro, article outline, short video script, image prompt. | — |
| Tone | The selectable voice applied across all formats: Professional, Casual, or Bold. | style, mood |
| Generation | The single action that expands one idea into all six formats. | render, conversion |
| Card | The editable UI block that displays one generated format. | panel, box |
| History | The last 10 ideas saved in localStorage for reload. | log, saved items |
| Image prompt | A descriptive prompt for an image model (Midjourney / DALL-E) — one of the six formats, distinct from the Claude prompt that drives generation. | — |
